# 战斧F1手柄驱动程序
严格意义来说并不是驱动。只是一个键位映射工具，因为战斧F1手柄有意地将手柄按照非标准的Android和Xinput手柄来设计，所以这个手柄直接用在Android或Windows上，键位是乱的。且Windows平台上不支持扳机（LT、RT）的线性。
此工具是基于这个项目的代码修改而来的：https://github.com/irungentoo/Xiaomi_gamepad
使用了SCP作为模拟X360的驱动。
